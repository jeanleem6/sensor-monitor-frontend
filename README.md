# 设备监测系统说明

## 一、第一层：（楼栋）

### 楼栋数据

1. 全参数气象站（温度、湿度、露点、风向、风速、气压、降水量、二氧化碳、PM2.5雾霾、PM10灰尘）
2. 太阳辐射观测系统（直接辐射、散射辐射、光谱辐射）
3. 摄像头（区域滞留人数、进入人数、离开人数、经过人数）点击查看视频
4. 手持及其它设备数据（设备总数）
5. 突发报警

### 楼栋数据排列

- 中间：楼栋模型
- 左边：
  1. 全参数气象站
  2. 太阳辐射观测系统
- 右边：
  1. 摄像头
  2. 手持及其它设备数据（设备总数）- 点击查看所有设备
- 下边：
  1. 突发报警

## 二、第二层：（楼层）

### 楼层数据

1. 温度传感器
2. 湿度传感器
3. 二氧化碳传感器
4. 电表-空调能耗
5. 电表-照明和插座
6. 水表-用水量
7. 红外线传感器

### 楼层数据排列

- 中间：楼层模型
- 左边：
  1. 各种设备总数（温度、湿度、二氧化碳、红外线传感器）：包含在线、离线
  2. 能耗监测设备总数：包含在线、离线
  3. 能耗监测总量：电表-空调能耗、电表-照明和插座能耗、水表-用水量能耗
- 右边：
  1. 温度和湿度所有房间数据
  2. 二氧化碳所有房间数据
  3. 红外线传感器所有房间数据

## 三、第三层：（房间）

### 房间数据

1. 温度传感器
2. 湿度传感器
3. 二氧化碳传感器
4. 电表-空调能耗
5. 电表-灯光插座
6. 红外线传感器

### 房间数据排列

- 中间：房间模型
- 左边：
  1. 房间电表-空调能耗（点击查看历史数据）
  2. 房间电表-灯光插座（点击查看历史数据）
  3. 房间红外线传感器数据（点击查看历史数据）
- 右边：
  1. 房间温度数据（点击查看历史数据）
  2. 房间湿度数据（点击查看历史数据）
  3. 房间二氧化碳数据（点击查看历史数据）
