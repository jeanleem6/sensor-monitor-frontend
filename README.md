# 设备监测系统说明

## Online Website

http://47.92.89.38:8066/

## 一、第一层：（楼栋）

### 楼栋数据

1. 全参数气象站（温度、湿度、露点、风向、风速、气压、降水量、二氧化碳、PM2.5雾霾、PM10灰尘）
2. 太阳辐射观测系统（直接辐射、散射辐射、光谱辐射）
3. 摄像头（区域滞留人数、进入人数、离开人数、经过人数）点击查看视频
4. 能耗·水耗·环境
   1. 能耗相关
      1. 今日用电量 kWh
      2. 当前用电负荷 kW
      3. 本周峰值用电负荷 kW
      4. 单位面积能耗 kWh/m²
      5. 碳排放量kg
      6. 用能费用 元
   2. 水耗相关
      1. 今日用水量 m3
      2. 当前用水负荷 m3/h
      3. 用水费用 元
   3. 环境相关
      1. 目前楼层停留总人数（通过人流摄像机统计得到）
      2. 平均温度、湿度、CO2浓度
5. 突发报警

### 楼栋数据排列

- 中间：楼栋模型
- 左边：
  1. 全参数气象站
  2. 太阳辐射观测系统
- 右边：
  1. 摄像头
  2. 能耗·水耗·环境
- 下边：
  1. 突发报警

## Nginx 服务器配置

```nginx
server {
    listen       {端口};                   # 如: 80
    server_name  {域名/IP/localhost};

    location / {
       root   {站点路径};                  # 比如: E:/wwwroot/cq-ust
       try_files $uri $uri/ /index.html;  # 重要：必须保留
    }
}
```
